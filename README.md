  ## Prerequistes
This project uses many frameworks and NPMs. Use <a href="https://nodejs.org/en">npm</a> to install these packages
<ul>
<li>Express</li>
</ul>

```bash
npm install express
```
<ul>
<li>dotenv/li>
</ul>

```bash
npm install dotenv
```
<ul>
<li>mongoose</li>
</ul>

```bash
npm install mongoose
```
<ul>
<li>cors</li>
</ul>

```bash
npm install cors
```

<ul>
<li>body-parser</li>
</ul>

```bash
npm install body-parser
```
